Week 4 Class:

Layouts and design systems:

  • Box Models:
    • Include the padding, spacing within a "box", and how you arrange elements within a box.
    • Flex boxes allow for for flexibility when styling text within a box.
  • Flexboxes:
    • When we apply attributes to flex boxes it will only affect the children and not the grandchildren.
    • Main Access: If it is a row, the main access that we will be working on will be on the x-axis.
    • Cross Acess: If it is a column, the axis that we will be working on will the the y-axis.
  • Grid Boxes:
    • grid-template-columns: 1fr1fr1fr1fr; or Fraction Units.
      • Fraction Units work in ratios, if something is "grid-template-column: 1fr 2fr;" it means that the second column is twice as thick was the previous column.
