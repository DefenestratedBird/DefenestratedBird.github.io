Dom & Information Architecture:

  • Document Object Models (DOMS) are like the blueprint of your house/website.
  • It shows which parts of the house/websites are interactable.
    • The door, the windows, etc.
  • There are 3 top level Doms in Javascript.
    1. Window.
    2. Document
    3. Navigator
  • QuerySelectors:
    • The search bar of Javascript that shifts through your html code for tags & elements.
      • Example: QuerySelector(p);

Reading Dom Elements:

  • Getter:
    • Getters gets the element.property & or "content" of an Element.
  • Setters:
    • Setters changes the element.property/"content" to a new value.

Adding or Removing Classes:

  • myClasses.classList.add("new-class");
    • This command tells your Javascript to add a new style.
    • Quote on Quote it is telling your Javascript to dress your HTML in a new outfit.
  • myClasses.classList.remove("old-class");
    • This tells your Javascript to remove a old style from your HTML.
    • Javasript: "REMOVE THINE OLD OUTFIT!"
    • HTML: "Ok."
  • myClasses.classList.toggle("class-name");
    • This command tells your Javascript to remove or add a style depending on whether it's already there.
